PASOS PARA ACTUALIZAR:

PARA BORRAR
git rm <archivo: Elimina el archivo del directorio de trabajo y prepara la eliminación para el siguiente commit.
git rm -f <archivo: Fuerza la eliminación si el archivo ya tenía cambios en el área de preparación (staging).
git rm --cached <archivo: Elimina el archivo del índice (repositorio), pero lo mantiene en tu equipo local.
